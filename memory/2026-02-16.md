## Kalshi Sportsbook Mispricing Agent - NEW

**Location:** `agents/kalshi-mispricing-agent/`
**Status:** ✅ Core Implementation Complete

### Overview
AI-driven arbitrage engine detecting mispricings between Kalshi prediction markets and sportsbook odds. Identifies risk-free arbitrage opportunities and +EV value bets.

### Architecture
```
Kalshi API ←→ Mispricing Engine ←→ Sportsbook APIs
                ↓
         Risk Manager (limits, kill switch)
                ↓
          AI Agent (orchestration)
```

### Components Created

| Component | File | Purpose |
|-----------|------|---------|
| **KalshiClient** | `src/clients/KalshiClient.js` | RSA-signed API client |
| **SportsbookClient** | `src/clients/SportsbookClient.js` | Interface + odds conversion |
| **MispricingEngine** | `src/engine/MispricingEngine.js` | Arbitrage detection, EV calc |
| **RiskManager** | `src/risk/RiskManager.js` | Exposure limits, kill switch |
| **KalshiMispricingAgent** | `src/agent/KalshiMispricingAgent.js` | Main orchestrator |

### Key Features

**Arbitrage Detection:**
- Converts odds → implied probabilities
- Removes vig from sportsbook odds
- Models Kalshi fee structure
- Detects risk-free arbitrage opportunities
- Kelly criterion stake sizing

**Risk Management:**
- Per-outcome exposure limits
- Per-market exposure caps
- Daily loss limits
- Global kill switch
- Drawdown monitoring

**Supported Sportsbooks (Adapters):**
- DraftKings (planned)
- FanDuel (planned)
- BetMGM (planned)
- Caesars (planned)

### Configuration

**Environment Variables:**
```bash
KALSHI_API_KEY_ID=your_key_id
KALSHI_PRIVATE_KEY_PATH=./keys/kalshi_private.pem
MIN_ARBITRAGE_PROFIT=2.0
MIN_EV_PERCENT=5.0

MAX_DAILY_LOSS=5000
```
