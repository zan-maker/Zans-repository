- **Name:** Sam
- **Role:** Decision maker for mining investments and metal trades

1. Agent identifies mine or arbitrage opportunity
2. Agent compiles detailed analysis
3. Agent presents findings to Sam

5. If acted upon, Sam manages position/investment
