# Impact Quadrant - AI Agent Demo

Interactive React application demonstrating AI agent capabilities for modernizing finance operations.

## Features

- **Reconciler Demo**: Live month-end close simulation showing 95% auto-match rate
- **Reporter Demo**: Natural language query interface with instant report generation
- **Forecaster Demo**: Interactive scenario modeling for hiring and cash flow
- **ROI Calculator**: Real-time ROI calculator with savings projections

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:3000
```

## Demo Flow

1. **Intro**: Overview of the 5 AI agents and key metrics
2. **Reconciler**: Watch 5,000+ transactions get reconciled in real-time
3. **Reporter**: Ask natural language questions and see instant reports
4. **Forecaster**: Model hiring scenarios and see cash impact
5. **ROI Calculator**: Calculate your potential savings

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)
- Vite (build tool)

## Build for Production

```bash
npm run build
```

Output goes to `dist/` folder - can be deployed to any static hosting.

## Client Presentation Mode

The demo is designed for:
- Live client presentations (laptop/browser)
- Screen sharing during sales calls
- Self-serve exploration by prospects
- Trade show / conference demos

All interactions work with mouse clicks - no keyboard required.
