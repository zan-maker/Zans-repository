# COMPONENT NOTES FOR DEVELOPERS

## Typography Hierarchy
- **H1 (Hero):** 48-64px, bold, tight line-height
- **H2 (Section Headers):** 36-48px, bold
- **H3 (Subsection):** 24-32px, semibold
- **Body:** 16-18px, regular, comfortable line-height (1.6)
- **Micro-copy:** 14px, regular, muted color

## Tone Checkpoints
- âœ“ Conversational but professional
- âœ“ Active voice throughout
- âœ“ Numbers and specifics included
- âœ“ Outcome-focused, not feature-focused
- âœ“ Headlines under 10 words
- âœ“ Minimal jargon

## Animation Opportunities
1. Hero: Orbiting agent orbs with pulsing effects
2. Problem/Solution: Interactive before/after slider
3. Agents: Cards with hover states showing "active" mode
4. How It Works: Timeline with scroll-triggered progress
5. Metrics: Count-up animation for before/after numbers

## Accessibility Notes
- All icons need aria-labels
- Color contrast minimum 4.5:1 for body text
- Animated sections need pause controls
- Keyboard navigable agent cards
