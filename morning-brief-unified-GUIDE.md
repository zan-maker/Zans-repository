## How to Use This Template

1. **Copy** this template to create each day's brief
2. **Replace** {{VARIABLES}} with actual data
3. **Customize** sections based on what's relevant
4. **Save** as `briefs/YYYY-MM-DD.md`

## Variables to Replace

| Variable | Description | Example |
|----------|-------------|---------|
| {{DATE}} | Today's date | 2026-02-11 |
| {{TIME}} | Generation time | 7:30 PM |
| {{X}} | Numeric values | 3 opportunities |
| {{Event Name}} | Specific market/event | "Fed Rate Decision" |
| {{Mine Name}} | Mining property name | "Gold Creek Project" |
| {{Company Name}} | Lead company name | "TechStart Inc." |

## Output Format

Each agent should populate their section:
- TradeRecommender: Trading table + key insight
- MiningMetalsAnalyst: Deal alerts + database summary  
- LeadGenerator: Scored leads by tier

## Cross-Opportunity Section

Look for connections:
- Mining company with CFO job posting
- Trading opportunity in sector with active lead
- Lead company in commodity-related industry

## Performance Tracking

Log metrics to track over time:
- Recommendations per agent
- Confidence score trends
- Alert frequency
- Action completion rates

## Archive Completed Briefs

Move to `briefs/archive/` after 30 days.
Keep latest 30 days in `briefs/` for reference.
