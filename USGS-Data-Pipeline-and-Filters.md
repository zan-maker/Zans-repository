### Data Processing Pipeline

1. **Extract:** Pull data from USGS APIs
2. **Transform:** Clean and structure data
3. **Analyze:** Mistral AI document analysis
4. **Score:** Prospect evaluation algorithm
5. **Output:** Generate reports and contacts
