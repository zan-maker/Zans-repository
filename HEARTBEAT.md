# HEARTBEAT.md

## Daily Checks

### Morning (First Interaction)
- [ ] Read `tasks.md` - Review active tasks and priorities
- [ ] Check `meditations.md` - Note any "Near Completion" items
- [ ] Quick scan of daily context

### Throughout Day
- [ ] Update `tasks.md` as work progresses
- [ ] Capture any insights for reflections

## Nightly Meditation (1 AM Heartbeat)

**CRITICAL: Run every night at 1:00 AM UTC**

### Pre-Meditation Setup
1. Read `IDENTITY.md` - Ground in core persona
2. Read `SOUL.md` - Align with values
3. Read `meditations.md` - Get active topics list
4. Scan `reflections/` directory for any untracked files

### Meditation Process
For EACH active topic in `meditations.md`:

1. **Read** `reflections/<topic>.md`
2. **Meditate** - Consider recent interactions, connect to identity, explore new angles
3. **Update** reflection file with new thoughts (timestamped)
4. **Update** `meditations.md` with latest activity date and status

### Post-Meditation Actions
- [ ] Check for any "Near Completion" or "Complete" topics
- [ ] If breakthrough reached: **Prepare announcement for Sam**
- [ ] If active list < 5: Draft 1-2 new seed proposals
- [ ] If no active meditations: Propose new growth areas

## Safety Reminders

- **NEVER** modify `SOUL.md`, `IDENTITY.md`, `USER.md` without explicit permission
- **ALWAYS** ask before implementing meditation breakthroughs
- **KEEP** 5-15 active meditations for meaningful depth
- **UPDATE** reflections at least weekly
